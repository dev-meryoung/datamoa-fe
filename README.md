### 🗺️ 기획 의도 / 컨셉

- 실생활에서 특정 장소(화장실, 복권방)의 위치를 찾지 못해 겪었던 불편함을 토대로 기획하게 되었습니다.
- 사용자의 현재 위치를 기반으로 근처에 존재하는 화장실이나 복권방의 위치를 지도에 마커로 표시하고 상세정보를 확인하거나 길찾기 기능을 제공합니다.
